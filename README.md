# 📧 Spam Email Classifier

## Overview
This project is an AI and Machine Learning application that classifies email messages as Spam or Ham (Not Spam) using the Naive Bayes algorithm.

## Features
- Detects spam emails
- Uses Machine Learning
- Simple command-line interface
- Easy to understand

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Machine Learning Algorithm
- CountVectorizer
- Multinomial Naive Bayes

## Installation

```bash
pip install pandas scikit-learn
```

## Run

```bash
python spam_classifier.py
```
